# RAG-PDFs

Un assistant qui permet de poser des questions à partir de documents PDF.

## Fonctionnalités
- Indexer un PDF et créer des embeddings
- Poser des questions et obtenir des réponses basées sur le contenu
- API simple avec Express pour upload et chat
- Interface simple avec HTML et CSS pour discuter et poser des question 

## Installation
1. `npm install`
2. Créer `.env` avec vos clés API
3. `npm start` pour lancer le serveur
